### 특이한 정렬

def solution(numlist, n):
    numlist.sort(key = lambda x: (abs(x-n), -x))
    
    return numlist

- numlist와 sort를 풀면 이렇게도 할 수 있음.
- abs는 절대값으로 무조건+3으로 만드는 짓임.

리스트명.sort(key=lambda x: (1순위_기준, 2순위_기준, ...))
key= : "그냥 크기 비교하지 말고, 내가 지금부터 주는 특별한 규칙으로 정렬해!"

lambda x: : "리스트에서 원소를 하나씩 꺼내서 x라고 부르고 공식에 넣을게!"

( , ) : 괄호 안에 쉼표로 기준을 나열하면 "1순위가 비기면(동점이면) 2순위로 채점해!"라는 뜻 (3순위, 4순위도 계속 추가 가능)
