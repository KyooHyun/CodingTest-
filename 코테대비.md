# 복합 기준 (1순위: 첫번째 오름차순, 2순위: 두번째 오름차순)
arr.sort(key=lambda x: (x[0], x[1]))
# → [[1, 'a'], [1, 'b'], [2, 'a']]

# 내림차순은 - 붙이기 (숫자만 가능)
arr.sort(key=lambda x: (-x[0], x[1]))


⚠️ 문자열 내림차순은 - 못 씀 → reverse=True 사용

4. 그리디 (Greedy)

회의실 배정 패턴


핵심: 끝나는 시간 기준 정렬 → 앞에서부터 선택



pythondef solution(meetings):
    meetings.sort(key=lambda x: x[1])  # 끝나는 시간 기준
    count = 1
    end = meetings[0][1]

    for i in range(1, len(meetings)):
        if end <= meetings[i][0]:       # 현재 끝 <= 다음 시작
            count += 1
            end = meetings[i][1]        # end 업데이트

    return count


5. 이중 포인터 (Two Pointer)

pythondef solution(arr, target):
    left = 0
    right = len(arr) - 1

    while left < right:
        s = arr[left] + arr[right]
        if s == target:
            return True
        elif s > target:
            right -= 1   # 합이 크면 right 줄이기
        else:
            left += 1    # 합이 작으면 left 늘리기

    return False



arr[left] + arr[right] > target → right 줄이기
arr[left] + arr[right] < target → left 늘리기
